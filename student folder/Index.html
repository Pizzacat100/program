Who should take the Web & Mobile Development Program?

People looking for a new career

Web Developers are well paid, according to indeed Web Developers make on average of $60,000 a year in the state of Utah. Most of our students leaving the program make about $16-$24 an hour, or between $33,000 to $49,000 a year upon graduation. Pay usually increases after 1 year of experince.

Entrepreneurs or freelancers

Wheather you want to learn how to make a website for your company or you want to freelance and make websites for others, the Web & Mobile Development program can help you make your dream a reality. Many business owners don't have the budget to pay a Web Developer, the program will teach you how. As a freelancer, your income is only limited by how much you are willing to put into your business. Furthermore; there is nothing quite as rewarding as working on something you have built!

Current developers looking to improve their skills

Are you currently employed as a junior or entree level developer and you want to improve your skills to get a better job? We can help you move beyond the basics so you can qualify for better employment!

Code hobbyist

It does not matter if you are interested in blogging or just want to learn how to code. We can teach you how to develop an online presence and learn to code for fun! Many of us who code find writing code to be enjoyable and even therapeutic

What will you learn?

HTML or Hypertext Markup language is used to structure websites

CSS or Cascading style sheets are used to format the layout of web pages

Databases are structured sets of data that can be accessed, managed and updated

PHP is and HTML embedded scripting language that can be used to access the server

Javascript is an object-oriented language used to create interactive effects on a webpage

Mobile or Responsive design is building websites to work on both desktops and mobile devices
